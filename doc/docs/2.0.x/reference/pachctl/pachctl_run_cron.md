## pachctl run cron

Run an existing Pachyderm cron pipeline now

### Synopsis

Run an existing Pachyderm cron pipeline now

```
pachctl run cron <pipeline> [flags]
```

### Examples

		# Run a cron pipeline "clock" now
		$ pachctl run cron clock
