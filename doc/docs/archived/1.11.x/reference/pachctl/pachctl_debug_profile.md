## pachctl debug profile

Return a profile from the server.

### Synopsis

Return a profile from the server.

```
pachctl debug profile <profile> [flags]
```

### Options

```
  -d, --duration duration   Duration to run a CPU profile for. (default 1m0s)
  -h, --help                help for profile
```
